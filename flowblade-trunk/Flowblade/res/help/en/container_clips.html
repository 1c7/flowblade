Container Clips

A Container Clip is a Media Item and the Timeline Clip made from it that has three items packed together as a single entity: Program, Unrendered Media Clip and Rendered Media Clip.

Container Clips Concept

The point of Container Clip can be a bit different for every type of container clip, there will be use case example for each type of Container Clip below.

Container Clips Components

Every Container Clip has the followiing three components:

  1. Unrendered Media. This is what is displayed on timeline if container clip media is not rendered.
  2. Program This data is used to to render media clip on timeline, sometimes using unrendered media as input.
  3. Rendered Media This what is displayed on timeline when Container Clip is rendered.

Rationale

  • Performance Container Clips provide means to turn resource heavy media items such as Sequence Clips into rendered media
  • Resource Management E.g. Blender Projects and G'Mic scripts are nicely managed by having them available via Container Clips
  • Capablity extension Existing technologies and applications can made to provide functionality that is not available otherwise. There wil be more developments in this area in the future.
  • Sharing In the future we will create a central repository to share container clip resources.

Workflow

  1. Create Container Clip Media Item using Project -> Create Contaniner Clip menu.
  2. Edit properties of Container Clip with Right Mouse menu selection Edit Container Data if available.
  3. Add Container Clip on Timeline.
  4. Render and manage Container Clips on Timeline with Right Mouse submenu Container Clip Actions.

Container Clips Types

We currently have 3 different types of Container Clips:

  1. G'Mic Script Container Clips.
  2. Blender Project Container Clips.
  3. MLT XML Container Clips.

G'Mic Script Container Clips

Components

  • Unrendered Media Video Clip
  • Program G'Mic script created and saved from G'Mic tool
  • Rendered Media A clip rendered with G'mic effect

Use Cases
  • Render G'Mic effects only on clip ranges used on timeline.
  • Make easier to use single userdefined G'Mic effect on multple clips.

Blender Project Container Clips

Components

  • Unrendered Media A placeholder Video Clip rendered on Media item creation.
  • Program Project file with *blend extension.
  • Rendered Media Rendered Clipfrom project file.

Use Cases
  • Manage Blender projects by having handles for them in Flowblade.
  • Create editors for Blender projects so that users can edit their properies on Flowblade
  • Make possible to contribute and shear interesting Blender Projects for e.g. so nice text effects.

MLT XML Container Clips

Components

  • Unrendered Media A MLT XML vdieo clip created from selection or sequence like Compound clips previously
  • Program A MLT XML video clip cerated from selection or sequence like Compound clips previously. Here Program is the same as unrendered media
  • Rendered Media Clip Rendered from MLT XML Clip.

Use Cases
  • Make possible to create Media items from Slections and full Sequences.
  • Give better timeline performance for complex multitrack container clips by pre-rendering them on timeline.Container Clip

These are the same as previous Compound Clips. We dropped Compound Clip nomenclature and call these Container Clips now.
